# The Prob

## Controls:

movement -> wasd

jump -> space, J, C

light attack -> left click or K

heavy.attack -> L

dash -> ;, W, I

The green line is the health, which appears when taking damage.

The blue line is the mana of the player, which is needed to attack and make different moves.

## The Backstory:

Proby wants to go and complete this dungeon since this dungeon has a big treasure, and he wants to get it out. Also, the prob is a magical creature that has many abilities, like the light attack, which is a melee attack, and the heavy blow or attack, which is a projectile attack.

## The Goal:

Defeat all the monsters in the dungeon to get the score to go to the next level. After all this, you will get out of the cave or the dungeon.
