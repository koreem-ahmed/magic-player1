# The Prob
The Prob is a soulless gosht whose had its soul away so help him in his journey to get his soul back.
## the demo
https://koreem-ahmed.itch.io/the-prob

## Controls:
movement -> arrows

jump -> C

light attack -> x

heavy.attack -> f

dash -> z

The health & mana bars are on the top left the player needs mana to preform attacks or to dash the flighing flames restore some health and mana when collected with the player also the player needs 3 points on the top right counter to be able to preform the spell whch makes him able to teleport using the portal in the level.

## The Backstory:

Prob wants to go and complete this dungeon since this dungeon has a big treasure, and he wants to get it out. Also, the prob is a magical creature that has many abilities, like the light attack, which is a melee attack, and the heavy blow or attack, which is a projectile attack. So use this powers and collect the magical flames to resture mana and health.f

## The Goal:
	For each level kill 3 monsters to be able to teleport to the next level.
	After winning 3 levels you will get the prop's soul back.
